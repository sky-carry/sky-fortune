> 汇总自四份专项调研（2026-07-02）：[01-国内竞品调研](01-国内竞品调研.md) · [02-海外产品调研](02-海外产品调研.md) · [03-命理领域知识](03-命理领域知识.md) · [04-商业模式与合规](04-商业模式与合规.md)

## 一、五个核心结论

1. **技术上没有黑魔法**。中式命理功能本质是「历法换算 + 查表规则 + 文案库」。真正的计算难点（节气时刻、日柱、行星位置）都有成熟开源库：lunar-javascript/lunar-python（八字+黄历全套，MIT）、iztro（紫微斗数，MIT）、Swiss Ephemeris（西占，注意 AGPL/商业双许可）。**80% 的工作量在解读文案的内容工程，不在算法。**

2. **成熟玩家的共同结构是三层漏斗**：免费工具引流（排盘/抽签/测名）→ 付费报告变现（28~98 元/份）→ 真人咨询做利润（100 元+/次）。海外对应版本是：免费排盘 → 订阅制（$9-15/月）→ 按分钟真人咨询。

3. **境内合规是硬约束，不是姿态问题**。有偿算命 = 网信办下架关停先例（天天吉历）；微信小程序连免费算命内容都拒审；微信支付/支付宝不给这类业务开商户号；广告法禁投。境内能长活的只有「传统文化工具 + 起名服务 + 电商」形态。**出海（AI 八字 + 订阅制 + Stripe）是唯一被验证能正大光明做付费测算的路线**（FateTell 等案例，数百万美元级收入）。

4. **AI 是最大产品机会，但不能裸接 LLM**。60% 年轻人用过 AI 算命，需求已验证；但成功产品（Co-Star、AI Ra）的共同架构是：**确定性排盘引擎（保证算得对）→ 结构化解读语料库（保证口径统一）→ LLM 只负责组装个性化语言和对话**。差异化在命理引擎和语料，不在"接了 GPT"。

5. **市场足够大、用户画像清晰**：国内玄学消费超 1100 亿元/年，全球占星 App 市场 47.5 亿美元、~20% 年增速。主力用户是 16-35 岁女性（测测 80%、FateTell 60%），低客单（年预算多 <500 元）、高频、情绪驱动，"水逆/不顺"是消费触发点，获客渠道在小红书/B站/抖音/TikTok。

## 二、功能规划（按优先级）

### P0 — 核心闭环（MVP）

| 功能 | 输入 | 引擎/数据 | 角色 |
|---|---|---|---|
| 八字排盘 + 五行分析 | 生日+时辰+性别（+出生地） | lunar 库 + 十神/喜用神规则层 | 核心引擎，一切功能的地基 |
| 八字精批报告（性格/事业/财运/婚姻/大运） | 同上 | 排盘结果 → 解读文案库（+LLM 组装） | 核心变现品 |
| 起名 + 姓名测试打分 | 姓氏+性别+生日 / 姓名 | 五格三才 + 81 数理 + 八字补缺 + 生肖宜忌（全查表） | 刚需流量入口 + 高客单（268 元起），合规风险最低 |
| 八字合婚 / 生肖配对 | 双方生日 | 干支关系查表 + 打分 | 转化率最高的付费品类，天然裂变 |
| 每日运势（生肖/星座） | 生日 | 流年太岁查表 + 内容运营/LLM 批量生成 | 留存与召回 |

### P1 — 扩充流量面

- **在线抽签**（观音签/月老签，随机数+签文库，成本最低、仪式感设计是重点）
- **老黄历/择日**（lunar 库开箱即用，工具属性最强、合规最安全，可做境内主入口）
- **称骨算命、周公解梦、测字、手机号/车牌吉凶**（长尾 SEO 标配，全是查表+文案）
- **塔罗**（78 张牌义 JSON 开源，抽牌动画 + LLM 个性化解读，抓年轻女性用户）
- **AI 命理问答**：基于用户已排的命盘做限定语料问答，按次卖提问包（Co-Star Void 模式，做成仪式感入口而非普通聊天框）

### P2 — 深度与差异化

- 紫微斗数排盘 + 详批（iztro 现成）
- 西方占星本命盘（需 Swiss Ephemeris，注意许可；海外版价值高）
- 好友合盘/关系档位（The Pattern 的 soulmate→challenging 分档，社交裂变）
- 占卜日记/命盘收藏（数据沉淀 = 留存资产）
- 六爻/梅花易数（硬核用户粘性）

### 明确不做

- 真人大师付费咨询撮合（测测 3·15 前车之鉴）
- 分销代理裂变、私域引流卖"开运消灾"商品
- AI 面相/手相拍照识别（生物特征信息，PIPL 高危）
- 任何"消灾、改运、保佑"类承诺性话术

## 三、关键决策：市场定位（需要拍板）

| 路线 | 玩法 | 收入模式 | 风险 |
|---|---|---|---|
| **A. 出海**（推荐主战场） | 英文/繁中站，AI 八字+紫微，面向海外华人和欧美东南亚玄学用户 | 订阅 $9-15/月 + 单次报告 $20-130，Stripe 收款，可投广告 | 竞争者增多，但空间大且合法 |
| **B. 境内文化工具站** | 黄历/择日/节气为主体 + 起名服务 + 免费娱乐测试 | 广告 + 起名（268-3898 元档）+ 文创电商 | 不能碰付费测算；表述要去承诺化 |
| **C. 境内付费测算站** | 三藏/卜易居式 | 付费命书 28-98 元 | **随时可能被下架关停，不建议** |

技术上 A 和 B 可以共享同一套排盘引擎和内容库，分开域名、主体和支付。

## 四、架构建议

```
┌─ 前端（网站优先：SEO 流量 + 绕开应用商店/小程序审核 + Web 支付无抽成）
├─ 内容层：解读文案 CMS（十神×宫位×格局的结构化语料，词条页做 SEO）
├─ AI 层：LLM 基于命盘 JSON + 限定语料生成个性化报告/对话（境内需 AIGC 备案+标识）
├─ 规则引擎：十神/喜用神/五格/合婚/神煞——纯查表，表驱动 + 单元测试
└─ 历法引擎：lunar-javascript（八字/黄历）+ iztro（紫微）+ Swiss Ephemeris（西占，可选）
```

- **回归测试**：命理流派分歧多（真太阳时、晚子时、旺衰权重），选定基准后与元亨利贞/问真八字逐例比对，典型生日做成测试集。
- **数据资产采集清单**：康熙笔画字典、汉字五行表、81 数理批语、三才吉凶表、生肖宜忌部首、称骨表+骨歌、诸葛 384 签、灵签文案、64 卦、塔罗牌义、十神/星曜解读语料（注意现代解签文本版权）。
- **产品设计基调**：去神秘化的"新中式"审美（参考 Co-Star 的智性极简，避免大红大金）；心理学化语言（The Pattern）；仪式感交互（抽牌动画、排盘等待动效）；可分享的命格卡片做社交货币。
